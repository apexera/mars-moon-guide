======================
Product Brief 
======================

NetBot is based on MT7620A Soc, designed by MedeaTek for wireless router and iot solution. The basic features as below:

===========      ===================================
===========      ===================================
Soc          MT7620A@600MHz
Memory      256MB
Flash       SPI Flash 16M
USB         4xUSB Hub
Ethernet    1xRJ45
Wifi        builtin AP
LED         7xLed, GPIOs
Uart        1 Debug Uart
Power
RTC         NXP PCF8563
===========      ===================================
